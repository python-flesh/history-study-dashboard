# 雪山书房

面向中央民族大学中国史方向考研的本地复习打卡页面。

## 使用方式

在当前目录运行：

```bash
python3 -m http.server 4173
```

然后在浏览器中打开：

```text
http://localhost:4173
```

## 数据说明

- 每日任务、打卡状态和周复盘保存在当前浏览器的 `localStorage` 中。
- 页面不需要账号，也不会把记录上传到服务器。
- 可以在“周复盘”页面导出 JSON 备份，换电脑或清理浏览器前建议先导出。

## 分享给朋友

### 直接发送文件夹

将整个文件夹压缩后发给朋友。对方解压后，双击 `index.html` 即可使用。

### 发布为网址

这个项目是纯静态网页，不需要构建。可以将整个文件夹拖放到
[Netlify Drop](https://app.netlify.com/drop)，部署完成后把生成的
`netlify.app` 地址发给朋友。

每个人的数据仍然只保存在各自的浏览器中，互不影响，也不会自动同步。
